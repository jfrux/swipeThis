.swipeThis()

swipeThis is a jQuery / jQuery mobile plugin for easily creating swipeable lists of items.

It is prepared to work on mobile devices and desktop browsers, you can have one or multiple items visible, items can be clickable / tapable, and much more..

Dependencies

jQuery 1.5.x

Simple Demo

Demo of Multiple Items with links

Using swipeThis

Import the scripts 

	    <script type="text/javascript" src="/js/jquery.1.6.1.min.js"></script> 
	    <script type="text/javascript" src="/js/swipeThis.0.7.3.min.js"></script>

The Markup 


    <ul id="myGallery">
        <li><img src='img/photo1.jpg'></li>
        <li><img src='img/photo2.jpg'></li>
        <li><img src='img/photo3.jpg'></li>
    </ul>

The style

&lt;style  type="text/css"&gt;
      .swipeThis {
        overflow:hidden;
      }
      .swipeThis ul {
        list-style:none;
        position:relative;
        margin:0px;
        padding:0px;   
      }
      .swipeThis li {
         display:block;
         float:left;   
      }
      .swipeArea {
        position:absolute;
        z-index:30;
        background-image:url('img/trans.png');
	cursor:move;
    }
&lt;/style&gt;

Swipe It!

    $('#MyGallery').swipeThis();

Customize it! 

    $('#myGallery').swipeThis({
        multiple:true,
        marginRight:15,
        indicator:'/img/swipeIndicator.png'});

List of configuration Options:

  • Multiple (default: false)
  • fillContainer (default: true)
  • fullScreen (default: false)
  • itemWidth (default: 170)
  • itemHeight (default: 140)
  • indicator : Image url for the icon that tells users to swipe to see more content. (default: '')
  • marginRight: Margin applied to items in Multiple mode (default: 0)
  • speed: Transition speed (default: 100)
  • easein: jQuery easeIn function (default: '')
  • tolerance: (default: 0.5)
